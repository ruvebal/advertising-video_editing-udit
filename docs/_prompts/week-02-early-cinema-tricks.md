### Technical Workshop: Stop-Motion & Substitution Cuts (30 minutes)

#### Demonstration: Creating a Substitution Cut

**Instructor demonstrates live:**

**Setup:**
- Place object (e.g., coffee cup) on table
- Frame shot with phone/camera on tripod (or stable surface)
- **Critical:** Camera MUST NOT MOVE

**Technique 1: Substitution Cut (MÃ©liÃ¨s Style)**

1. **Shot 1:** Film object for 3 seconds (cup on table)
2. **Stop camera** (do NOT move camera position)
3. **Remove object** (take cup away)
4. **Resume filming** for 3 seconds (empty table)

**In Premiere Pro:**
- Import both clips
- Place on timeline sequentially
- The "edit" is invisibleâ€”cup vanishes!

**Technique 2: Stop-Motion Animation**

1. **Frame 1:** Object in position A
2. **Take photo**
3. **Move object slightly** to position B
4. **Take photo**
5. **Repeat** 20-30 times for smooth motion

**In Premiere Pro:**
- Import all photos
- Set duration to 0.1 seconds each (10 frames per second)
- Photos play in sequence = animation!

**Pro Tip:** More photos = smoother motion. MÃ©liÃ¨s used fewer frames (jerky), ChomÃ³n used more (smoother).

#### Adobe Premiere Pro Techniques for Stop-Motion

**1. Importing Image Sequences:**
```
File > Import
Check "Image Sequence" box
Select first image in folder
Premiere imports all as video clip
```

**2. Adjusting Frame Duration:**
```
Right-click image sequence on timeline
Speed/Duration
Set to desired speed (slower = smoother)
```

**3. Frame-by-Frame Navigation:**
```
Arrow keys: Move one frame forward/back
Shift+Arrow: Move 5 frames
Page Up/Down: Move to edit points
```

**4. Precise Cutting:**
```
Zoom in timeline (+ key or scroll)
Use Razor tool (C) at exact frame
Or Cmd+K to cut at playhead
Frame accuracy is critical for invisible edits
```

**5. Speed Ramping (Advanced):**
```
Right-click clip > Show Clip Keyframes > Time Remapping > Speed
Add keyframes to vary speed
Slow-motion for emphasis, speed-up for comedy
```

#### Production Phase (30 minutes)

**Setup Stations:**
- Divide class into groups of 2-3
- Each group gets filming space (desk, corner, wall)
- Ensure stable camera positions (tripods, phone stands, or stacks of books)

**Filming Tips (Demonstrate):**

1. **Lock Your Camera:**
   - Use tripod or stable surface
   - Mark camera position with tape
   - If camera moves, effect fails!

2. **Consistent Lighting:**
   - Don't film near windows (light changes)
   - Use room lights consistently
   - Shadows reveal the trick if inconsistent

3. **Plan Your Movements:**
   - For stop-motion: Small, incremental movements
   - For substitution: Mark object positions with tape
   - Test a few frames first

4. **Shoot More Than You Need:**
   - Extra frames = options in editing
   - Can always cut, can't add what you didn't shoot

**Common Mistakes to Avoid:**
- Camera movement (ruins illusion)
- Inconsistent lighting (visible in edit)
- Too few frames (jerky motion)
- Forgetting to mark positions (continuity errors)

**Instructor Role:**
- Circulate between groups

- Celebrate creative ideas

## Community Best Practices: Stop-Motion & VFX

### From Professional Animators & VFX Artists

**1. "Animate in Layers"**
- Separate foreground, midground, background
- Easier to control and adjust
- Creates depth

**2. "Onion Skinning" (Reference Previous Frame)**
- In stop-motion apps: See ghost of previous frame
- Ensures smooth motion
- Premiere doesn't have built-in, but you can:
  - Duplicate previous frame on track above
  - Lower opacity to 30%
  - Use as reference for next position

**3. "The 12 Principles of Animation" (Disney)**
Even for simple stop-motion, key principles apply:
- **Squash and Stretch:** Exaggerate for life
- **Anticipation:** Wind up before action
- **Staging:** Clear silhouettes and composition
- **Ease In/Out:** Slow start, slow end (not linear motion)

**4. "Test Early, Test Often"**
- Shoot 10 frames, review
- Adjust before committing to 100 frames
- Saves time and frustration

**5. "Lighting Consistency is Everything"**
- Use artificial light (not windows)
- Mark light positions
- Shoot in one session if possible (sun changes)

**Honda "Paper" (2015):**
- Entire Honda history told through paper cutouts
