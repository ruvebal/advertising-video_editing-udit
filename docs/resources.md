## Community Best Practices

{% include tao-quote.html quote="The wise editor organizes before creating. Chaos in the project folder leads to chaos in the timeline." %}

### Project Organization
```
PROJECT_NAME/
├── 01_RAW/
│   ├── Video/
│   └── Audio/
├── 02_SEQUENCES/
├── 03_EXPORTS/
├── 04_ASSETS/
│   ├── Music/
│   ├── SFX/
│   └── Graphics/
└── 05_ARCHIVE/
```

### Naming Conventions
- Use descriptive names: `SCENE01_INTERVIEW_JOHN_V02.mp4`
- Include version numbers
- Use dates in ISO format: `2025-01-14`

### Backup Strategy
- 3-2-1 Rule: 3 copies, 2 different media, 1 offsite
